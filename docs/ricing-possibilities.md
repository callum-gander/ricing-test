# Ricing possibilities & roadmap

The full surface area of what can be customised on a Hyprland/Wayland desktop, as a
checklist. `[x]` = already done in this repo · `[~]` = partially done · `[ ]` = fair game.

Two things to remember at the top: this catalogue is the **floor, not the ceiling** —
(1) anything custom-built in Quickshell/AGS has no catalogue entry, and (2) the *reactive*
layer (wiring the look to live system state) is where it becomes genuinely bottomless.

---

## 0. Boot & login (before the desktop)
- [~] **Plymouth** — animated boot splash (enabled, but not visible in the VM; a metal thing)
- [x] Greeter: **tuigreet** in the VM; **ReGreet** wired and ready for metal (one-line enable)
- [x] Session autologin (greetd → Hyprland)
- [ ] Bootloader theme (GRUB / rEFInd; systemd-boot is plain)

## 1. The compositor — Hyprland
- [x] Animations (bézier curves; window / fade / workspace)
- [x] Decoration: rounding, gradient borders, shadows, blur, active/inactive opacity
- [x] Layout: dwindle (BSP)
- [x] Window rules: float/size/position per app (block syntax)
- [x] Scratchpad (special workspace, `Super+S`)
- [x] Plugins: **hyprbars** (title bars) + **hyprspace** (workspace overview). NOTE: upstream removed
      hyprexpo/hyprtrails as unmaintained — we use nixpkgs `hyprlandPlugins`, and hyprspace is the
      maintained overview. borders-plus-plus / hyprfocus / hy3 / hyprgrass also available there.
- [x] **Submaps** (resize submap; modal keybind layers)
- [~] Cursor: **Bibata** via `home.pointerCursor` (not yet animated hyprcursor SVG)
- [x] Screen shaders (GLSL): per-pixel vibrance/glow/vignette (`glow.frag`); real bloom (`bloom.frag`, metal)
- [ ] Tabbed window groups; gestures (3-finger swipe — no touchpad in the VM); `keyd` remapping

## 2. The shell / bar / widgets
- Framework: **Quickshell** (QML, current), **waybar** (classic), **AGS/Astal** (JS/TS), **eww**, **fabric**, nwg-shell
- [x] Bar: logo, live workspaces, clock, volume
- [x] Bar modules: **system tray**, network, bluetooth, **now-playing (MPRIS)**, CPU/RAM
- [x] Popups: **notification centre**, **control centre** (audio/net/bt/system, animated, click-away dismiss),
      **launcher**, **system monitor** (btop), **workspace overview** (hyprspace)
- [ ] More: window title, battery, brightness OSDs, temp, keyboard layout, updates, weather, album art, dock, calendar
- [ ] Desktop widgets: clock/date/weather overlays, conky-style info, audio visualiser

## 3. Wallpaper
- [x] Static gradient (swaybg, Mocha radial gradient generated by Nix)
- [ ] **awww** (animated, transitions, GIF), **mpvpaper** (video), shader wallpapers
- [ ] Per-workspace / per-monitor / time-of-day cycling; audio-reactive

## 4. Launcher & menus
- [x] **rofi** (default, themed) + **fuzzel** (installed)
- [ ] Alternatives: anyrun, walker, tofi, or Quickshell-custom
- [ ] Extra uses: calculator, emoji, **clipboard history** (cliphist), window switcher, power menu

## 5. Notifications
- [x] **swaync** (centre + history, default) + **mako** (installed)
- [ ] Quickshell-custom notification stack

## 6. Lock / idle / power
- [x] **hyprlock** (`Super+L`)
- [x] **hypridle** (idle → lock)
- [ ] **wlogout** (power-menu grid); blur/album-art/widgets on the lock screen

## 7. Terminal & CLI
- [x] Terminal: **Ghostty** (default) + **foot** fallback, both themed
- [x] Prompt: **starship**
- [x] Ricey CLI tools: **btop**, **fastfetch**, **eza/bat**, **lazygit**, **zellij**
- [ ] Shell: fish/zsh + plugins (currently bash); **cava** (audio visualiser); terminal CRT/glow shaders

## 8. Editor / dev workspace
- [x] **Neovim** (LazyVim) — themed, bootstraps on first launch
- [ ] noice.nvim / dashboard / transparency tuning; tmux/zellij themed status

## 9. Per-app theming (multiplies endlessly)
- [ ] **Firefox `userChrome.css`** (vertical tabs, hidden chrome), **Zen/Floorp**
- [ ] VSCode / Obsidian themes (both installed, not yet themed), **Stylus** (per-website CSS)
- [ ] **Spicetify** (Spotify), **Vencord** (Discord)

## 10. System-wide theming
- [x] Manual Catppuccin Mocha (foot/ghostty/rofi/mako/swaync/hyprland/quickshell/zellij/nvim/starship)
- [~] **Editorial Paper** — warm-light alt palette (sharp corners, hairline rules, rust accent,
      Playfair Display + Noto Sans, 一二三四 CJK workspaces). Prototyped on branch
      `theme/editorial-paper`; full write-up in `docs/status.md`.
- [x] Nerd Font (JetBrains Mono)
- [~] Cursor theme (Bibata). GTK/Qt themes (qt5ct/Kvantum), icon theme (Papirus/Tela) still open.
- [ ] **Stylix** (Nix) — one base16 palette auto-themes everything
- [ ] **Dynamic colour-from-wallpaper**: **matugen / wallust / pywal**
- [ ] Light/dark auto-switch (wlsunset)

## 11. Effects & motion
- [x] Hyprland screen shaders, animated borders, blur layers
- [ ] Mouse trails (hyprtrails was dropped upstream — needs an alternative), audio-reactive visuals

## 12. The reactive / scripted layer (the deep end)
- [ ] Hyprland **`socket2` events** → run scripts on window/workspace/focus changes
- [ ] State-driven visuals (accent tracks battery / CPU / time-of-day)
- [ ] **Wallpaper → palette pipelines** (matugen recolours everything live)
- [ ] Custom keybind macros; screenshot/clipboard workflows

## 13. Audio & hardware
- [ ] **EasyEffects** (EQ/effects, ricey UI); custom system sounds
- [ ] **OpenRGB** — sync peripheral lighting to the palette (real hardware only)

## 14. Utilities / workflow tools
- [x] Screenshot: grim + slurp (`Print`)
- [ ] **hyprshot / grimblast / satty**; **wf-recorder** / OBS; **hyprpicker**, **cliphist**, **wlsunset**

## 15. Functional rice (workflow, not just looks)
- [x] Scratchpad, resize submap, per-app window rules
- [ ] Quick-toggle panels; focus / DND automation; pomodoro / activity widgets

---

### The two bottomless dimensions
- **Custom-built components** — anything you can imagine in Quickshell/AGS (no catalogue).
- **Reactivity** — wiring the look to live system state so it feels *alive*, not static.
